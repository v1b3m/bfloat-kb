## Codebase Overview

**Bfloat IDE** (v0.2.10) is an AI-powered desktop IDE for building web and mobile apps with Claude and ChatGPT. It uses a **Tauri v2** shell with a **Bun+Hono sidecar** backend and a **React 19** frontend.

Auth: HTTP Basic (`bfloat:<uuid>` generated per launch by Tauri).

### AI Agent System

- **`lib/agents/`**: `DefaultAgentManager` manages three providers:
  - `claude` вҖ” `@anthropic-ai/claude-agent-sdk` (Claude Code CLI SDK)
  - `codex` вҖ” OpenAI Codex
  - `bfloat` вҖ” Direct Anthropic API (legacy pay-per-use)
- Session options: `cwd`, `model`, `permissionMode`, `allowedTools`, `systemPrompt`, `mcpServers`, `maxTurns`, etc.
- Stream frame types: `init`, `text`, `reasoning`, `tool_call`, `tool_result`, `error`, `done`, `stream_end`, `cancelled`

### Startup Sequence

1. Tauri boots вҶ’ spawns `bfloat-sidecar` binary on a random localhost port
2. Health-check loop polls `GET /health` every 100ms (30s timeout)
3. WebView loads вҶ’ `ServerGate` calls `await_initialization` to get `{url, password}`
4. `SidecarApi` singleton initialized вҶ’ `window.conveyor` compat shim installed
5. React app renders вҶ’ onboarding gate вҶ’ routes to Home/Project/Settings

### Product Routes

- `/` вҖ” Home: project browser + AI project creation
- `/settings` вҖ” Provider auth, appearance, preferences
- `/projects/:id` вҖ” Full IDE workbench (editor, preview, terminal, chat)

### Build & Dev

```bash
pnpm dev            # Tauri dev (Vite on :1420 + Rust shell + sidecar)
pnpm build          # Production build
pnpm build:sidecar  # Compile sidecar binary for current platform
pnpm lint           # ESLint
pnpm format         # Prettier
```

Requires: Node 20+, pnpm 10+, Rust вүҘ1.88.0, Bun.
