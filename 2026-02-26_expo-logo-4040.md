# Expo Logo 404 Fix

## Description
The Expo Go app icon in the QR code was returning 404, leaving a broken image in the center of the QR code.

## Progress
- [x] Identify the broken URL in Preview.tsx
- [x] Replace with local bfloat icon
- [x] Fix visibility (dark icon is white-on-transparent, invisible on white QR background)
- [x] Overlay icon with dark background instead of using imageSettings
- [x] Shrink overlay and bump error correction to H for scannability

## Decisions
- Used the dark (white) icon variant on a black rounded background — matches bfloat branding better than the colored light variant
- Switched from QRCodeSVG `imageSettings` to a CSS overlay approach for full control over the icon background
- Bumped QR error correction from M (15%) to H (30%) to compensate for center coverage
- Icon overlay sized at 17% of QR size — small enough to stay scannable

## Outcome
Replaced broken Expo icon URL with local bfloat dark icon overlaid on a dark background in the QR code center.
