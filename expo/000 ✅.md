For this session, we're goign to hack on automating deployments to expo and the apple app store

1. Review [https://launch.expo.dev/](https://launch.expo.dev/)
2. Look into `npx testflight`
3. Look into `eas` workflows
4. Connecting to app store connect
5. We need to leverage mostly all of the expo tooling locally so that users can easily deploy
6. Since we have access to a terminal, users will simply click a button and we shall run all the necessary commands on their behalf in a shell session.
   1. It should ideally be interactive cause some of these flows (like apple) will require users to click links to navigate to the apple developer console (website) and then input tokens(codes) into the terminal
7. There's a "Deploy" button on the project page, clicking this button should trigger a deployment
   1. Deploy should open the deployment modal
