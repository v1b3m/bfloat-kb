# Note Naming Convention

## Rule
All root notes in this vault must use:
- `YYYY-MM-DD-rest-of-title`

Example:
- `2025-12-13-rest-of-title`

## Guidelines
- Use the note creation date for `YYYY-MM-DD`.
- Use kebab-case for the remainder (`rest-of-title`).
- Avoid unnamed notes (`Untitled ...`).
- For duplicate titles on the same day, append `-2`, `-3`, etc.

## Scope
- Applies to root-level notes in `bfloat-kb`.
- Folders can keep their own naming conventions if needed.
